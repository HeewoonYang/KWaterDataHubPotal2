# K-water 데이터허브 포털 (DataHub Portal)

K-water 클라우드 데이터 에코시스템의 데이터허브 포털 웹 애플리케이션

## 시스템 구성

### 아키텍처

```
┌─────────────┐     ┌──────────────┐     ┌────────────┐
│  Frontend    │────▶│   Backend    │────▶│ PostgreSQL │
│  (Vue 3)    │     │  (FastAPI)   │     │  :5433     │
│  :8088      │     │  :8001       │     └────────────┘
└─────────────┘     └──────┬───────┘
                           │         ┌────────────┐
                           ├────────▶│   Redis    │
                           │         │   :6379    │
                           │         └────────────┘
                           │         ┌────────────┐
                           ├────────▶│  Celery    │
                           │         │  Worker    │
                           │         └────────────┘
                           │         ┌────────────┐
                           └────────▶│  Celery    │
                                     │  Beat      │
                                     └────────────┘
```

| 서비스 | 설명 | 포트 |
|--------|------|------|

## 포털 구조

본 시스템은 **사용자 포털**과 **관리자 포털** 2개의 독립적인 레이아웃으로 구성된다.

### 사용자 포털 (`/portal/*`)

상단 GNB + 서브탭 구조. 역할 기반 메뉴 접근 제어 적용.

| GNB 메뉴 | 서브메뉴 | 설명 |
|-----------|----------|------|
| **대시보드** | 대시보드 | 개인화 대시보드 (위젯 기반) |
| | 위젯 설정 | 대시보드 위젯 배치/설정 |
| | 시각화 갤러리 설정 | 갤러리 콘텐츠 설정 |
| | 위젯 관리 | 위젯 템플릿 관리 (관리자) |
| | 갤러리 콘텐츠 관리 | 시각화 콘텐츠 생성/관리 (D&D) |
| | 실시간 계측DB | 실시간 계측 데이터 조회 |
| **데이터카탈로그** | 카탈로그 탐색 | 데이터셋 탐색/상세 조회 |
| | 데이터 검색 | 통합 데이터 검색 |
| **데이터유통** | 유통 데이터 목록 | 유통 가능 데이터 조회 |
| | 데이터 신청 | 데이터 이용 신청 |

| **마이페이지** | 내 프로필 | 개인정보 조회/수정 |
| | 내 데이터 | 내가 등록/신청한 데이터 |

```
├── package.json
├── vite.config.ts
└── tsconfig.json
```

## 개발 환경 실행

```bash
# 의존성 설치
npm install

# 개발 서버 (HMR)
npm run dev          # http://localhost:5173

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

### Docker 실행 (전체 스택)

```bash
# 프로젝트 루트에서
docker compose up -d --build

# Frontend: http://localhost:8088
# Backend API: http://localhost:8088/api/v1/
```

## 디자인 규칙

- **사용자 포털**: `index.html` 디자인 기준 (상단 GNB + 서브탭)
- **관리자 포털**: `kwater_portal.html` 디자인 기준 (좌측 사이드바 + 상단 GNB)
- **아이콘**: `@ant-design/icons-vue`만 사용 (이모지, 타 라이브러리 금지)
- **반응형**: PC(1280px+), 태블릿(768px~1279px) 대응. 모바일 미지원.
